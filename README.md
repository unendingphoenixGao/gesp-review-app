# 年糕 GESP C++ 五级复习系统

这是一个本地网页复习系统，用来帮助复习 GESP C++ 五级。

## 怎么使用

1. 下载这个仓库。
2. 打开 `index.html`。
3. 先看“先复习”里的知识点，再进入“再答题”练习选择题。

## 内容

- 知识点复习卡片
- 选择题题库
- 已练题记录
- 错题练习
- 曾错题和订正状态
- 编程题知识点拆解

题库数据已经生成在 `data.js` 中，直接打开网页即可使用，不需要安装软件。

## 练习记录

答题记录会自动保存在当前电脑的浏览器里。下次用同一个浏览器打开 `index.html` 时，可以继续练未做题，也可以只练当前错题。

如果换一台电脑，练习记录不会自动带过去，需要重新开始。
